Advanced mode (modalità avanzata)

Fino ad ore abbiamo considerato esclusivamente le funzionalità più semplici di GameMaker, ma il programma offre molte altre possibilità. Per utililizzarle bisogna innanzitutto eseguire GameMaker in advanced mode. Questo é molto semplice da fare, basta selezionare dal menu File la voce Advanced mode (per visualizzare in modo completo gli effetti di questa modifica GameMaker va riavviato, o perlomeno il gioco salvato e ricaricato).

Quando GameMaker viene eseguito in advanced mode, si presenta come segue:

Contiene tutte le funzionalità della simple mode, ma con diverse risorse, bottoni e voci a menu addizionali. Inoltre, come vedremo nei prossimi capitoli, diverse risorse presenteranno delle opzioni in più. In questo capitolo tratteremo le voci a menu aggiuntive.

Menu File

Nel menu file è possibile trovare i seguenti comandi aggiuntivi:

  • Publish your Game (Pubblica il tuo gioco). Questo comando porta al nostro sito web dove è possibile caricare e pubblicare in modo semplice il tuo gioco completo in modo che chiunque possa giocarci. Segui attentamente le istruzioni per la pubblicazione. Ti preghiamo di utilizzare questa funzionalità solo per i giochi completi, e non per versioni preliminari. Questa funzionalità è disponibile solo nella Standard Edition
  • Import Resources (Importa Risorse). Questo comando ti permette di importare risorse esportate con il comando Export Resources, spiegato di seguito. Per informazioni dettagliate vedi sezione Importare ed esportare risorse.
  • Export Resources (Esporta Risorse). Questo comando permette di esportare risorse in un file. Per informazioni dettagliate vedi sezione Importare ed esportare risorse.
  • Preferences (Preferenze). Qui é possibile impostare le preferenze di GameMaker. Queste impostazioni verranno salvate dal programma e mantenute alle aperture successive. Per maggiori informazioni vedere la sezione Preferenze.

Menu Edit

Nel menu edit è possibile trovare i seguenti comandi aggiuntivi:
  • Add group (Aggiungi gruppo). Le risorse possono essere raggruppate. Questo é molto utile in giochi con molte risorse. Si può per esempio inserire tutti i suoni relativi a un certo oggetto in un gruppo, o raggruppare tutti gli oggetti presente in un particolare livello. Questo comando crea un nuovo gruppo nella tipologia di risorsa attualmente selezionata. Verrà richiesto un nome. I gruppi possono contenere altri gruppi, ecc. Le risorse possono essere trascinate da un gruppo all'altro.
  • Find Resource (trova risorsa). Questo comando richiede l'inserimento del nome di una risorsa della quale aprirà la relativa finestra di proprietà.
  • Expand Resource Tree (Espandi albero delle risorse). Espande completamente l'albero delle risorse, mostrandone tutti i contenuti.
  • Collapse Resource Tree (Chiudi albero delle risorse). Chiude completamente l'albero delle risorse, nascondendonei contenuti.
  • Show Object Information (Mostra informazioni oggetto). Con questo comando é possibile avere una panoramica degli oggetti di gioco e delle loro proprietà.

Menu Resources

In questo menu è possibile creare nuove risorse. Tenete presente che per ogni categoria di risorsa è presente un bottone anche nella toolbar e delle combinazioni di tasti. Sono presenti 3 voci particolari in fondo al menu:
  • Define Constants (Definisci costanti). Qui è possibile definire delle costanti da utilizzare negli script e nelle azioni. Per maggiori informazioni sulle costanti vedere sezione Costanti.
  • Define Triggers (Definisci triggers). Qui é possibile definire i propri triggers. Per maggiori informazioni sui triggers vedere sezione Eventi trigger.
  • Included Files (File inclusi). Qui é possibile indicare files che verranno inclusi nel proprio gioco. Per maggiori informazioni vedere sezione Includere Files.

Menu Scripts

Nel menu scripts è possibile trovare i seguenti comandi:
  • Import Scripts (Importa scripts). Può essere usato per importare script da files esterni.
  • Export Scripts (Esporta scripts). Può essere usato per salvare i propri scripts in un file, utilizzabile da altri. Quando solo uno script è selezionato, solo quest'ultimo viene salvato. Quando un gruppo di scripts è selezionato, tutti gli scripts in quel gruppo verranno salvati. Quando é selezionata la sezione base o un'altro tipo di risorsa, tutti gli scripts verranno salvati. Questa funzionalità è presente anche cliccando col tasto destro su uno script o un gruppo di scripts.
  • Show Built-in Variables (Mostra variabili built-in). Mostra una lista ordinata di variabili built-in, locali e globali
  • Show Built-in Functions (Mostra funzioni built-in). Mostra una lista ordinata di funzioni built-in.
  • Show Extension Functions (Mostra funzioni delle estensioni). Mostra una lista ordinata di tutte le funzioni disponibili negli extension packages inclusi nel gioco.
  • Show Constants (Mostra costanti). Mostra una lista ordinata di costanti sia built-in che definite dall'utente.
  • Show Resource Names (Mostra nomi risorse). Mostra una lista ordinata di tutti i nomi delle risorse. È possibile cliccare sul nome di una risorsa per aprire il relativo editor.
  • Search in Scripts (Cerca negli scripts). Permette di cercare una determinata parola chiave all'interno di tutti gli scripts. È possibile cliccare sul nome di una risorsa per aprire il relativo editor.
  • Check Resource Names (Controlla nomi risorse). Effettua un controllo sui nomi di tutte le risorse. Nomi non corretti, duplicati o in conflitto con variabili, funzioni o costanti, verranno segnalati. È possibile cliccare sul nome di una risorsa per aprire il relativo editor.
  • Check All Scripts (Controlla tutti gli scripts). Effettua un controllo su tutti gli scripts per errori. È possibile cliccare sui nomi riportati per modificare lo script.
