File INI

L’uso dei files INI è un meccanismo standard per passare determinati settaggi di parametri a dei programmi.I file INI contengono sezioni ed ogni sezione contiene una serie di coppie nome-valore. Ad esempio, ecco un tipico file INI: 

[Form]
Top=100
Left=100
Caption=Il miglior gioco di sempre
[Game]
MaxScore=12324

Questo file contiene due sezioni, una chiamata Form e l’altra chiamata Game. La prima sezione contiene tre coppie. Le prime due hanno un valore reale mentre la terza ha un valore stringa. Tali file INI sono facili da creare e modificare. Sono presenti le seguenti funzioni in GameMaker per leggere e modificare i dati in essi contenuti.

ini_open(name) Apre il file INI con il nome dato. Il file ini deve essere memorizzato nella stessa cartella del gioco!
ini_close() Chiude il file INI aperto.
ini_read_string(section,key,default) Legge il valore della stringa della chiave indicata nella sezione indicata. Quando la chiave o la sezione non esiste viene restituito il valore predefinito.
ini_read_real(section,key,default) Legge il valore reale della chiave indicata nella sezione indicata.Quando la chiave o la sezione non esiste viene resituito il valore predefinito.
ini_write_string(section,key,value) Scrive il valore della stringa per la chiave indicata nella sezione indicata.
ini_write_real(section,key,value) Scrive il valore reale per la chiave indicata nella sezione indicata.
ini_key_exists(section,key) Restituisce un valore se esiste la chiave indicata nella sezione indicata.
ini_section_exists(section) Restituisce un valore se esiste la sezione indicata.
ini_key_delete(section,key) Cancella la chiave indicata dalla sezione indicata.
ini_section_delete(section) Cancella la sezione indicata.
