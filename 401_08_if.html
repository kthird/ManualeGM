Costrutto If

Il costrutto If (if statement) ha questa forma: 

if (<espressione>) <azione>

oppure 

if (<espressione>) <azione> else <azione>

L'azione può anche essere un blocco di azioni. L'espressione verrà risolta. Se il valore (arrotondato) è <=0 (false(falso)) viene eseguita l'azione dopo else (altrimenti), in caso contrario (true(vero)) viene eseguita l'altra azione. È una buona abitudine mettere sempre entro parentesi graffe le azioni nel costrutto if. Perciò è meglio usare: 

if (<espressione>)
{
  <azione>
}
else
{
  <azione>
}

Esempio
Il programma seguente muove l'object verso il centro dello schermo: 

{
  if (x<200) {x += 4} else {x -= 4};
}
