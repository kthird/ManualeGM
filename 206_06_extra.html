Azioni extra

Nella tabella Extra ci sono anche un numero di azioni relative all'esecuzione di CD. Queste funzioni sono disponibili solo nella versione Standard di GameMaker.

Play CD
Con questa azione puoi esegure delle traccie da un CD nel lettore CD predefnito. Specifica la traccia iniziale e quella finale.

Stop CD
Blocca l'esecuzione corrente del CD.

Pause CD
Ferma l'esecuzione corrente del CD.

Resume CD
Riprende un CD in pausa.

Check CD
Se c'è un CD nel lettore CD predefinito esegue la prossima azione.

Check CD Playing
Se un CD è in esecuzione nel lettore CD predefinito eseque la prossima azione.

Infine, c'è un azione addizionale che può essere utile per certi giochi.

Set Cursor
Puoi usare questa azione pe sostituire il cursore del mouse della finestra con uno sprite. Specifica lo sprite e se il cursore di windows deve essere ancora mostrato. Lo sprite può essere animato. Nota che lo sprite sarà mostrato solo all'interno della room non all'infuori di essa.
