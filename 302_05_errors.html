Opzioni sugli Errori

Qui puoi impostare delle opzioni relative al modo in cui vengono riportati gli errori.

Display error messages
Significa "Mostra i messaggi d'errore". Quando è attivata, i messaggi d'errore vengono mostrati. Nella versione finale del gioco vorrai disattivare questa opzione.

Write error messages to file game_errors.log
Significa "Registra i messaggi d'errore nel file game_errors.log". Quando è attivata, tutti i messaggi d'errore vengono scritti in un file chiamato game_errors.log, che si trova nella cartella del gioco.

Abort on all error messages
Significa "Termina con tutti i messaggi d'errore". Normalmente, alcuni errori sono fatali mentre altri possono essere ignorati. Attivando questa opzione tutti gli errori verranno consideratio fatali e termineranno il gioco. Nella versione finale del tuo gioco, quando lo pubblicherai, vorrai attivare questa opzione.

Treat uninitialized variables as 0
Significa "Considera le variabili non inizializzate come 0". Un errore comune è usare una variabile prima che le venga assegnato un valore. A volte, tuttavia, ciò è difficile da evitare. Attivando questa opzione tali variabili non inizializzate non riporteranno più un errore ma verranno considerate come 0. Fa' attenzione, però. Significa che potresti non accorgerti dei tuoi errori di battitura.
