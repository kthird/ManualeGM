Splash Screens

Questa funzionalità è disponibile solo nella Standard Edition di GameMaker.

Molti giochi hanno i cosiddetti "splash screen". Sono delle schermate che mostrano un video, un'immagine o del testo. Spesso sono usati all'inizio del gioco (come introduzione), all'inizio di un livello, o alla fine del gioco (ad esempio per i titoli di coda ). In GameMaker questi splash screen con testo, immagini o video possono essere mostrati in ogni momento durante il gioco.

L'impostazione predefinita fa comparire gli splash screen nella finestra del gioco, ma è anche possibile mostrarli in una finestra separata. Il gioco viene interrotto mentre viene mostrato lo splash screen. Il giocatore può tornare al gioco premendo Esc o cliccando col mouse sulla finestra. (Puoi cambiare tutte queste impostazioni, vedi sotto.)

Le seguenti funzioni possono essere usate per mostrare gli splash screen:

splash_show_video(fname,loop) Mostra un video in uno splash screen. fname è il nome del file video. Un file specifico può essere supportato o no a seconda dei driver installati sul computer. Di solito puoi usare i file .avi, .mpg e .wmv ma evita codifiche speciali. Farai bene a mettere tu stesso quel file nella cartella del gioco o in una sottocartella. loop può essere true o false e indica se il video deve essere ripetuto ciclicamente o no. In qquest'ultimo caso lo splash screen si chiuderà automaticamente alla fine del video.
splash_show_text(fname,delay) Mostra del testo in uno splash screen. fname è il nome del file di testo. Puoi mostrare file di testo semplici (.txt) o file di testo arricchito (.rtf). Solo parte del testo arricchito viene presa in considerazione. Ad esempio non verranno mostrate immagini o altri oggetti incorporati. delay indica il numero di millisecondi che devono passare prima di tornare automaticamente al gioco. Usa 0 o un valore negativo per visualizzare lo splash screen per un tempo indefinito (il giocatore dovrà premere Esc o cliccare sulla finestra per tornare al gioco). Nota che non ci sarà modo di scorrere il testo, quindi devi farlo entrare nella finestra.
splash_show_image(fname,delay) Mostra un'immagine in uno splash screen. fname è il nome del file immagine. Sono supportati molti tipi di immagine (ad esempio .bmp, .jpg, .tif e .wmf) ma non puoi usare immagini animate. delay indica il numero di millisecondi che devono passare prima di tornare automaticamente al gioco.
splash_show_web(url,delay) Mostra una pagina web in uno splash screen. url è l'url della pagina (comincia con http://) oppure il nome di un file html salvato localmente. Nota che devi fornire il persorso completo per farlo funzionare. Quindi farai bene a usare qualcosa come working_directory + "\index.html". delay indica il numero di millisecondi che devono passare prima di tornare automaticamente al gioco.

Puoi cambiare il modo in cui gli splash screen vengono visualizzati usando la funzioni qui sotto:

splash_set_main(main) Indica se lo splash screen deve essere mostrato nella finestra del gioco (true, predefinito) o in una finestra separata (false).
splash_set_scale(scale) Imposta il fattore di scala da usare quando mostri un video o un'immagine in uno splash screen. Se usi il valore 0 il fattore di scala viene scelto in modo da riempire la finestra (predefinito).
splash_set_cursor(vis) Imposta a true per mostrare il cursore nello splash screen (predefinito), imposta a false per nasconderlo. NON puoi nasconderlo se nello splash screen c'è un video.
splash_set_color(col) Imposta il colore dell'area che circonda l'immagine o il video.
splash_set_caption(cap) Imposta la didascalia della finestra dello splash screen. Ha effetto solo se stai usando una finestra separata per lo splash screen. L'impostazione predefinita è la stringa vuota.
splash_set_fullscreen(full) Indica se la finestra dello splash screen è a schermo intero (true) o no (false, predefinito). Ha effetto solo se stai usando una finestra separata per lo splash screen.
splash_set_border(border) Indica se la finestra dello splash screen ha un bordo (true, predefinito) o no (false). Ha effetto solo se stai usando una finestra separata per lo splash screen e quella finestra non è a schermo intero.
splash_set_size(w,h) Imposta la dimensione della finestra dello splash screen. Ha effetto solo se stai usando una finestra separata per lo splash screen e quella finestra non è a schermo intero. L'impostazione predefinita è 640x480.
splash_set_position(x,y) Imposta la posizione della finestra dello splash screen. Ha effetto solo se stai usando una finestra separata per lo splash screen e quella finestra non è a schermo intero. Windows determina la posizione predefinita.
splash_set_adapt(adapt) Indica se la dimensione della finestra deve essere adattata a immagini e video scalati (true, predefinito) o no (false). Ha effetto solo se stai usando una finestra separata per lo splash screen e quella finestra non è a schermo intero.
splash_set_top(top) Indica se la finestra dello splash screen deve stare sempre sopra tutte le altre (true, predefinito) o no (false). Ha effetto solo se stai usando una finestra separata per lo splash screen e quella finestra non è a schermo intero.
splash_set_interrupt(interrupt) Indica se il gioco deve fermarsi mentre viene mostrata la finestra dello splash screen (true, predefinito) o no (false). Ha effetto solo se stai usando una finestra separata per lo splash screen.
splash_set_stop_key(stop) Indica se bisogna chiudere lo splash screen quando il giocatore preme Esc (true, predefinito) o no (false).
splash_set_stop_mouse(stop) Indica se bisogna chiudere lo splash screen quando il giocatore clicca col mouse al suo interno (true, predefinito) o no (false).
splash_set_close_button(show) Indica se bisogna mostrare un pulsante di chiusura nell'angolo in alto a destra dello splash screen (true, predefinito) o no (fase). Torna utile se lo splash screen è a schermo intero o senza bordo.

C'è un tipo particolare di splash screen, ovvero le informazioni sul gioco che l'utente può inserire in GameMaker. Puoi mostrarle usando la funzione che si trova più in basso. Puoi anche caricare un file separato. Visualizzare le informazioni è come mostrare del testo in uno splash screen ma le impostazioni della visualizzazione vanno regolate mentre definisci le informazioni (Ctrl+O o secondo pulsante della barra degli strumenti dell'editor delle informazioni) e non puoi cambiarle usando le funzioni qui sopra. Se scegli di mostrarle in un'altra finestra, sappi che non è la stessa degli splash screen, quindi puoi mostrare le informazioni e uno splash screen contemporaneamente. Queste funzioni possono essere usate anche nella Lite Edition.

show_info() Mostra le informazioni del gioco.
load_info(fname) Carica le informazioni del gioco dal file chiamato fname. Dovrebbe essere un file di testo arricchito (.rtf). Con questa funzione puoi mostrare diverse informazioni in diversi momenti. Nota che, al contrario degli splash screen, questo file rtf non può contenere immagini.
