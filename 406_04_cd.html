Musica dei CD

Queste funzionalità sono disponibili soltanto nella standard edition di GameMaker.

Ci sono anche delle funzioni per avviare musica da CD.

cd_init() Deve essere chiamata prima di usare le altre funzioni. Deve essere chiamata anche quanto è stato cambiato un CD(o anche semplicemente di volta in volta).
cd_present() Restituisce quando un CD è presente nel lettore CD di default.
cd_number() Restituisce il numero di tracce del CD.
cd_playing() Restituisce quando il CD è in esecuzione.
cd_paused() Restituisce quando il CD è pausato o inattivo.
cd_track() Restituisce il numero della traccia corrente (1 = la prima).
cd_length() Restituisce la durata totale del CD in millisecondi.
cd_track_length(n) Restituisce la durata della traccia n del CD in millisecondi.
cd_position() Restituisce la corrente posizione nel CD in millisecondi.
cd_track_position() Restituisce la posizione corrente nella traccia in esecuzione in millisecondi.
cd_play(first,last) Indica al CD di eseguire le tracce da first a last. Per eseguire tutto il CD inserite 1 e 1000 come argomenti.
cd_stop() Arresta l'esecuzione.
cd_pause() Pausa l'esecuzione.
cd_resume() Riprende l'esecuzione.
cd_set_position(pos) Imposta la posizione nel CD in millisecondi.
cd_set_track_position(pos) Imposta la posizione nella traccia corrente in millisecondi.
cd_open_door() Apre lo sportello del lettore CD.
cd_close_door() Chiude lo sportello del lettore CD.

Cè una funzione molto generalizzata per accedere alle funzionalità multimediali di Windows.

MCI_command(str) Questa funzione invita una riga di comando al sistema multimediale di Windows usando il Media Control Interface (MCI). Restituisce la stringa restituita dal comando. Puoi usare questa funzione per controllare ogni sorta di dispositivo multimediale. Guardate la documentazione di windows per informazioni su come usare questo comando. Per esempio MCI_command('play cdaudio from 1') avvia un CD (dopo averlo correttamente inizializzato usando altri comandi). Questa funzione è solo per uso avanzato!
