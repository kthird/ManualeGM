Paths

Le seguenti funzioni ti daranno informazioni su un path:

path_exists(ind) Restituisce true se esiste un path con l'indice dato.
path_get_name(ind) Restituisce il nome (in forma di stringa) del path con l'indice dato.
path_get_length(ind) Restituisce la lunghezza del path con l'indice dato.
path_get_kind(ind) Restituisce il tipo di linee usato nel path con indice dato (0=linee dritte, 1=curve smussate).
path_get_closed(ind) Restituisce true se il path è chiuso e false se non lo è.
path_get_precision(ind) Restiruisce la precisione usata per creare path smussati.
path_get_number(ind) Restituisce il numero di punti utilizzati per definire il path.
path_get_point_x(ind,n) Restituisce l'ascissa dell'n-esimo punto che definisce il path. 0 è il primo punto.
path_get_point_y(ind,n) Restituisce l'ordinata dell'n-esimo punto che definisce il path. 0 è il primo punto.
path_get_point_speed(ind,n) Restituisce il fattore di velocità dell'n-esimo punto che definisce il path. 0 è il primo punto.
path_get_x(ind,pos) Restituisce l'ascissa alla posizione pos del path. L'argument pos deve essere compreso tra 0 e 1.
path_get_y(ind,pos) Restituisce l'ordinata alla posizione pos del path. L'argument pos deve essere compreso tra 0 e 1.
path_get_speed(ind,pos) Restituisce il fattore di velocità alla posizione pos del path. L'argument pos deve essere compreso tra 0 e 1.
