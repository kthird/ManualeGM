Suoni e Musica

Molti giochi hanno degli effetti sonori e della musica di sottofondo. Alcuni effetti sonori utili possono essere trovati nel sito web di GameMaker. Altri possono essere trovati in altri posti sul web.

Per creare una risorsa suono (un sound) nel tuo gioco, usa l'elemento Create Sound nel menu Resources oppure usa il tasto corrispondente nella barra degli strumenti. La seguente finestra si aprirà.

Per caricare un suono, premi il bottone etichettato Load Sound. Si aprirà una finestra per selezionare il file. Ci sono due tipi di file sonori, i files wave e midi. I files wave vengono usati per degli effetti sonori brevi. Usano molta memoria, ma vengono riprodotti immediatamente. Usali per tutti gli effetti sonori del tuo gioco. I files midi descrivono la musica in un modo differente. Come risultato essi usano molta memoria in meno, ma sono limitati alla musica di sottofondo strumentale. Inoltre, può essere riprodotto solo un file midi alla volta.

Una volta caricato il file midi puoi ascoltarlo premendo il tasto play. C'è anche il bottone Save Sound per salvare il suono attuale in un file. Questo bottone non è realmente necessario ma potresti averne bisogno nel caso che tu perda il suono originale.
