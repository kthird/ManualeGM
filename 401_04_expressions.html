Espressioni

Le espressioni possono essere numeri reali (es.3.4), numeri esadecimali, preceduti dal segno $ (es. $00FFAA), stringhe messe tra apici singoli o doppi (es. 'hello' o “hello”) o espressioni più complicate. (Nota che le stringhe possono stare su più righe!) Per le espressioni, esistono i seguenti operatori binari (in ordine di priorità):
  • && || ^^: combinano valori Booleani (true/false) (&& = and, || = or, ^^ = xor)
  • < <= == != > >=: confronti, risultano true (1) o false (0)
  • | & ^: operatori bitwise (| = bitwise or, & = bitwise and, ^ = bitwise xor)
  • << >>: operatori bitwise (<< = shift left, > > = shift right)
  • + -: addizione, sottrazione
  • * / div mod: moltiplicazione, divisione, divisione intera e resto

Nota che il valore di x div y è il valore di x/y arrotondato per difetto all'intero più vicino. L'operatore mod Restituisce il resto ottenuto dividendo i suoi operandi. In altre parole, x mod y = x - (x div y) * y. Esistono anche i seguenti operatori unici:

  • !: not (non), trasforma true in false e viceversa
  • -: nega il prossimo valore
  • ~: nega il prossimo valore bitwise
Come valori puoi usare numeri, variabili, o funzioni che restituiscono un valore. Le sub-espressioni possono essere messe tra parentesi. Tutti gli operatori funzionano coi numeri reali. Le comparazioni finzionano anche per le stringhe, e il + può concatenare le stringhe. (Nota che, a differenza di altri linguaggi, entrambi gli argomenti di un'operazione Booleana sono sempre calcolati, anche se il primo determina già il risultato.)

Esempio

Ecco degli esempi di assegnazioni. 

{
  x = 23;
  color = $FFAA00;
  str = 'hello world';
  y += 5;
  x *= y;
  x = y << 2;
  x = 23*((2+4) / sin(y));
  str = 'hello' + " world";
  b = (x < 5) && !(x==2 || x==4);
}
