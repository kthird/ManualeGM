Sprites

Le seguenti funzioni ti daranno informazioni su uno sprite:

sprite_exists(ind) Ritorna quando uno sprite con il dato indice esiste.
sprite_get_name(ind) Ritorna il nome dello sprite con il dato indice.
sprite_get_number(ind) Ritorna il numero di sottoimmagini dello sprite con il dato indice.
sprite_get_width(ind) Ritorna la larghezza dello sprite con il dato indice.
sprite_get_height(ind) Ritorna l'altezza dello sprite con il dato indice.
sprite_get_xoffset(ind) Ritorna l'x-offset dello sprite con il dato indice.
sprite_get_yoffset(ind) Ritorna l'y-offset dello sprite con il dato indice.
sprite_get_bbox_left(ind) Ritorna il lato sinistro del bounding box dello sprite con il dato indice.
sprite_get_bbox_right(ind) Ritorna il lato destro del bounding box dello sprite con il dato indice.
sprite_get_bbox_top(ind) Ritorna il lato superiore del bounding box dello sprite con il dato indice.
sprite_get_bbox_bottom(ind) Ritorna il lato inferiore del bounding box dello sprite con il dato indice.
In certe situazioni potresti voler salvare il bitmpap corrispondente ad una particolare sottoimmagine dello sprite su un file. Per questo la seguente funzione può essere usata:

sprite_save(ind,subimg,fname) Salva la sottoimmagine subimg dello sprite ind al file con il nome fname. Questo deve essere un file .png. Disponibile solo nell'edizione Standard.
sprite_save_strip(ind,fname) Salva tutte le sottoimmagini dello sprite ind come una strip al file con il nome fname. Questo deve essere un file .png. Disponibile solo nell'edizione Standard.
