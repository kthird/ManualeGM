Funzioni basilari del Suono

Ci sono cinque funzioni basilari relativa al suono. Ce ne sono due per far partire un suono, una per controllare se un suono sta venendo riprodotto e due per fermare i suoni. La maggior parte prende come argument l'indice del suono.Il nome di un suono rappresenta il suo indice. Puoi anche salvare l'indice in una variabile e usare quella.

sound_play(index) Riproduce il suono indicato una volta. Se è una musica di sottofondo, quella corrente viene fermata.
sound_loop(index) Riproduce il suono indicato, ripetendolo all'infinito. È il cosiddetto looping. Se il suono è una musica di sottofondo, quella corrente viene fermata.
sound_stop(index) Ferma il suono indicato. Se ci sono più suoni con quest'indice in riproduzione, verranno fermati tutti.
sound_stop_all() Ferma tutti i suoni.
sound_isplaying(index) Restituisce true se (una copia di) il suono indicato è in riproduzione. Nota che questa funzione ti dice se il suono sta venendo emesso degli altoparlanti. Il suono non raggiunge gli altoparlanti subito dopo la chiamata della funzione per far partire un suono, quindi la funzione potrebbe ancora restituire false per qualche istante. Analogamente, quando fermi un suono lo senti ancora per un po' (ad esempio a causa dell'eco) e la funzione restituirà true ancora per qualche istante.

È possibile usare effetti sonori avanzati. In particolare puoi cambiare il volume e il "pan", cioè se il suono esce dall'altoparlante destro o sinistroe con quale intensità. In tutti qiesti casi il volume può solo essere ridotto. Queste funzioni non possono essere usate per i file che vengono riprodotti attraverso il media player (come gli mp3).

sound_volume(index,value) Cambia il volume del suono indicato (0 = nullo, 1 = massimo).
sound_global_volume(value) Cambia il volume globale per tutti i suoni (0 = nullo, 1 = massimo).
sound_fade(index,value,time) Cambia il volume del suono indicato nel nouvo valore (0 = nullo, 1 = massimo) modificandolo progressivamente in modo che ci metta time millisecondi. Può essere usato per far cominciare o terminare i suoni in maniera più elegante, facendoli sfumare cominciando o finendo allo 0.
sound_pan(index,value) Cambia il "pan" per il suono indicato (-1 = tutto o sinistra, 0 = equilibrato, 1 = tutto a destra).
sound_background_tempo(factor) Cambia il tempo della musica di sottofondo (se è un file midi). factor indica il fattore per cui va moltiplicato il tempo reale del suono. Quindi 1 corrisponde al tempo reale, valori maggiori a un tempo più veloce e valori minori a un tempo più lento. Questo fattore deve essere compreso tra 0.01 e 100.

Oltre al file midi e "wave" (e agli mp3) c'è in verità un quarto tipo di file che può essere riprodoto: i file di musica diretta (file.sgt). Spesso questi file si riferiscono ad altri file che contengono informazioni riguardanti, ad esempio, l'artista o il genere. Per trovare questi file, il sistema deve sapere dove sono. A questo scopo puoi usare la seguente funzione per impostare la directory di ricerca per i file. Tuttavia, nota che dovrai aggiungere questi file da solo. GameMaker non include automaticamente questi file addizionali.

sound_set_search_directory(dir) Imposta la directory in cui si trovano i file di musica diretta. La stringa dir non deve includere il backslash (\) finale.
