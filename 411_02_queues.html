Code

Una coda è molto simile ad una pila ma è basata sul metodo FIFO (First In First Out - il primo ad entrare è anche il primo ad uscire). Il valore che per primo viene inserito nella coda è anche il primo ad essere rimosso. Le code funzionano come quelle dei supermercati: la persona che è prima in una coda è servita per prima. Le code sono in genere utilizzate per memorizzare cose che non sono state ancora fatte, ma ci sono anche molti altri usi. Sono disponibili le seguenti funzioni (si noti che le prime cinque sono equivalenti alle funzioni delle pile; tutte le strutture dati hanno queste cinque funzioni in comune):

ds_queue_create() Crea una nuova coda. La funzione restituisce un intero come un id che deve essere utilizzato in tutte le altre funzioni per accedere a code particolari. È possibile creare più code.
ds_queue_destroy(id) Distrugge la coda dato il suo id, liberando la memoria utilizzata. Non dimenticate di chiamare questa funzione quando si è pronti con la struttura.
ds_queue_clear(id) Cancella la coda dato il suo id, rimuovendo tutti i dati da essa, ma senza distruggerla.
ds_queue_copy(id,source) Copia il contenuto della coda in un’altra coda dato il suo id.
ds_queue_size(id) Restituisce il numero di valori memorizzati nella coda.
ds_queue_empty(id) Restituisce un valore se la coda è vuota. È come testare se la dimensione è 0.
ds_queue_enqueue(id,val) Inserisce un valore nella coda.
ds_queue_dequeue(id) Restituisce il valore a capo della coda e lo rimuove dalla coda.
ds_queue_head(id) Restituisce il valore a capo della coda, e non lo rimuove dalla coda.
ds_queue_tail(id) Restituisce il valore alla fine della coda, cioè, il valore che più recentemente è stato aggiunto alla coda. (E non lo rimuove dalla coda).
ds_queue_write(id) Converte la struttura-dati (data structure) in una stringa e restituisce questa stringa. La stringa può quindi essere utilizzata, per esempio, per essere salvata in un file. Questa funzione fornisce un semplice meccanismo per salvare strutture dati.
ds_queue_read(id,str) Legge la struttura dati dalla stringa passata come parametro (che può essere creata usando la chiamata precedente).
