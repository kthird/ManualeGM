Definire gli sprite

Gli sprites sono la rappresentazione visiva di tutti gli oggetti nel gioco. Uno sprite è sia una singola immagine, disegnata con qualsiasi programma si preferisce, o un set di immagini che, quando eseguite una dopo l’altra, assomigliano ad una animazione. Per esempio, le seguenti quattro immagini formano uno sprite di un personaggio che si muove a destra.

Mentre crei un gioco solitamente inizi collezionando un set di sprite per gli oggetti nel tuo gioco. Un considerabile numero di sprite è compreso con GameMaker. Altre collezioni di interessanti sprite possono essere trovate sul sito di GameMaker. Altre ancora nel web, normalmente nel formato png o gif animate.

Per creare uno sprite, seleziona l'oggetto Create Sprite dal menu Resources, o utilizza il pulsante corrispondente nella toolbar. Il segunte menù dovrebbe apparire.

In alto a sinistra potrai indicare il nome dello sprite. Tutti gli sprite (e tutte le risorse) hanno un nome. La miglior cosa è dare un nome che descrive lo sprite. Devi far sì che tutte le risorse abbiano nomi diversi. Anche se questo non è strettamente richiesto, vi consigliamo di usare per il nome dello sprite solo lettere e numeri e il simbolo underscore (_) (vale anche per le altre risorse) preferibilimente iniziando con una lettera. In modo particolare non usate il carattere spazio. Questo diventerà importante una volta che inizierete ad usare il codice.

Per caricare uno sprite, cliccate sul bottone Load Sprite. Una finestra per file speciali verrà aperta, dove potrete scegliere lo sprite:

La parte sinsitra sembra come un normale selettore di file dove puoi selezionare lo sprite che vuoi. A destra puoi avere un anteprima dello sprite animato e alcune informazioni su di esso. Qui potete inoltre indicare se volete rendere lo sprite opaco (questo, rimuove tutte le parti trasparenti), se volete rimuovere il background, se volete renderlo trasparente (default), e se volete arrotondare gli angoli dello sprite, che può migliorare la sua apparenza. Quando sei contento del risultato, premi Open per caricare lo sprite.

GameMaker può caricare diversi tipo di file grafici. Quando caricate una gif animata, le diverse subimmagini formano le immagini degli sprite. Quando i nomi dei file finiscono con _stripXX, le XX sono un numero, si ritiene che contenga la striscia di XX subimmagini una dopo l'altra (non per i file GIF). Per esempio, un file immagine con il nome ball_strip4.png verrà contata avente 4 subimmagini.

Una volta che lo sprite è caricato la prima subimmagine verrà mostrata a destra. Quando ci sono più subimmagini, potrai cambiarle con le freccette.

Con i bottoni Edit Sprite potete modificare gli sprite, o creare un nuovo sprite. GameMaker ha una finestra di creazione sprite e un editor di immagini. Per più informazioni guardare le sezioni su Modificare i tuoi sprite e Modificare le subimmagini individuali.
