Paths

E’ possible creare dei path e aggiungere dei punti ad essi. Comunque non cambiate mai un path che è attualmente in uso da un'istanza. QUesto può portare ad errori inaspettati. Esistono le seguenti funzioni per la modifica dei path:

path_set_kind(ind,val) Setta il tipo di connessioni delle curve del path con il dato indice (0= spigolose, 1 = morbide).
path_set_closed(ind,closed) Imposta se il path deve essere chiuso (true) o aperto(false).
path_set_precision(ind,prec) Imposta la precisione con la quale calcolare le curve morbide del path (dovrà essere tra 1 e 8)
path_add() Crea un nuovo path vuoto e ne restituisce l'indice.
path_delete(ind) Elimina il path con il dato indice.
path_duplicate(ind) Crea una copia del path indicato e ne restituisce l'indice.
path_assign(ind,path) Assegna il path all'indice indicato. Questo creerà una copia del path. In questo modo puoi impostare facilmente un path esistente in un'altro, per esempio uno nuovo.
path_append(ind,path) Aggiunge il path indicato al path con indice ind.
path_add_point(ind,x,y,speed) Aggiunge un punto al path con il dato indice alla posizione (x,y) e con la velocità speed indicata. Ricordate che un valore di 100 corrisponde alla velocità attuale. Valori più bassi faranno rallentare mentre valori più alti faranno accelerare.
path_insert_point(ind,n,x,y,speed) Inserisce un punto al path con il dato indice alla posizione (x,y) prima del punto n con la velocità indicata.
path_change_point(ind,n,x,y,speed) Modifica il punto n nel path con il dato indice indicando posizione (x,y) e velocità.
path_delete_point(ind,n) Elimina il punto n del path con il dato indice.
path_clear_points(ind) Elimina tutti i punti del path ottenendo un path vuoto.
path_reverse(ind) Inverte il path.
path_mirror(ind) Specchia il path orizzontalmente (rispetto al suo centro)
path_flip(ind) Specchia il path verticalmente (rispetto al suo centro)
path_rotate(ind,angle) Ruota il path in senso antiorario con l'angolo indicato in gradi (rispetto al suo centro)
path_scale(ind,xscale,yscale) Scala il path con i valori indicati (rispetto al suo centro)
path_shift(ind,xshift,yshift) Sposta il path con i valori indicati.
