Script

Quando si crea uno script, si desidera accedere agli argomenti (o parametri) passati ad esso (sia quando si sta scrivendo lo script, sia quando lo si chiama come una funzione dal programma, o da un altro, o anche dallo stesso script - ricorsione -). Questi argomenti vengono memorizzati nelle variabili argument0, argument1, ..., argument15. Quindi ci possono essere al massimo 16 argomenti. (Si noti che quando si chiama lo script dallâ€™icona, solo i primi 5 argomenti possono essere specificati). Si possono anche usare argomenti di tipo array, come argument[0] etc.

Gli script possono anche restituire un valore, in modo che possano essere utilizzati nelle espressioni. A questo scopo si utilizza l'istruzione return: 

return <expression>

L'esecuzione di uno script termina con l'istruzione return!

Esempio
Ecco la definizione di un piccolo script che calcola il quadrato di un numero (argument0): 

{
  return (argument0*argument0);
}

Per chiamare uno script all'interno di un pezzo di codice si fa la stessa cosa di quando si chiamano le funzioni, ovvero, si scrive il nome dello script con i valori degli argomenti tra parentesi.
