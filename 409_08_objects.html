Oggetti

Anche gli oggetti possono essere manipolati durante la partita. NON cambiate o cancellate MAI un oggetto per il quale ci sono delle istanze. Questo può portate ad effetti inaspettati come certe proprietà dell’oggetto memorizzate con l’istanza e via dicendo, cambiandole nell’oggetto non si otterrà l’effetto desiderato.

object_set_sprite(ind,spr) Imposta lo sprite dell’oggetto con il dato indice. Usate -1 per rimuovere lo sprite attuale dall’oggetto.
object_set_solid(ind,solid) Imposta se le istanze create dell'oggetto devono essere solide di default (true o false)
object_set_visible(ind,vis) Imposta se le istanze create dell'oggetto devono essere visibili di default (true o false)
object_set_depth(ind,depth) Imposta la depth di default delle istanze create dell'oggetto.
object_set_persistent(ind,pers) Imposta se le istanze create dell'oggetto devono essere persistenti di default (true o false).
object_set_mask(ind,spr) Imposta la maschera dello sprite del'oggetto con il dato indice. Usate -1 per impostare che la mascherà sarà lo sprite dell'oggetto.
object_set_parent(ind,obj) Imposta il parent dell'oggetto. Usate -1 per non avere parent. Cambiare il parent cambia il comportamento delle istanze dell'oggetto.

Le seguenti funzioni sono utili per creare oggetti al volo. Come per tutte le altre funzioni per cambiare le risorse state attenti a non creare nuovi oggetti ogni volta.

object_add() Crea un nuovo oggetto e ne restituisce l'indice. Potete usare questo indice nelle funzioni qui sotto per impostare determinate proprietà dello'oggetto e per creare delle istanze.
object_delete(ind) Elimina l'oggetto con il dato indice. Assicuratevi di non avere istanze di questo oggetto attualmente esistenti.
object_event_add(ind,evtype,evnumb,codestr) Per cambiare il comportamento dell'oggetto bisogna definire degli eventi. A questi eventi si può aggiungere solo ed esclusivamente del codice. E' necessario specificare l'oggetto, il tipo di evento, il numero dell'evento(usate le costanti descritte prima della funzione event_perform()) ed infine la stringa contenente il codice che deve essere eseguito. Potete aggiungere più pezzi di codice per ogni evento.
object_event_clear(ind,evtype,evnumb) Usate questa funzione per eliminare tutte le azioni di un particolare evento.

Creare oggetti è particolarmente utile quando state progettando degli script o delle librerie di azioni. Per esempio, uno script di inizializzazione può creare un oggetto che mostri del testo e un’altro script può aggiungerlo ad un oggetto con un testo specifico. In questo modo avete un facile meccanismo per mostrare testi senza necessariamente create oggetti che usino l’interfaccia standard.
