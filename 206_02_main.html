Altre azioni principali

Alcune aggiuntive azioni principali sono disponibili nella modalità avanzata. Le seguenti azioni sono state aggiunte:

Set Time Line
Con questa azioni setti la particolare time line per un istanza di un oggetto. Indichi la timeline e la posizione iniziale entro la time line (0 è l'inizio). Indichi anche quando la time line dovrebbe iniziare immediatamente e quando dovrebbe ciclare alla fine.

Time Line Position
Con questa azione puoi cambiare la posizione nella time line attuale (o assoluto o relativo). Questo può essere usato per saltare certe parti della time line o ripetere certe parti.

Time Line Speed
Con questa azione puoi cambiare la velocità della timeline (o assoluta o relativa). Una velocità di 1 è la velocità normale. Quando ad esempio imposti a 2 le azioni saranno due volte più veloci e quando imposti a 0.5 due volte più lento. Puoi anche impostare una velocità negativa, in questo caso la timeline andrà all'indietro.

Start Time Line
Questa azione avvia una timeline dalla posizioneattuale (se è messa in pausa o interrotta).

Pause Time Line
Questa azione mette in pausa la timeline. Usa l'azione di inzio per lasciarla continuare.

Stop Time Line
Questa azione ferma la timeline e imposta indietro la posizione a 0.

Splash Text
Con questa azione puoi mostrare un file di testo. Questo può essere o un file .txt o un file .rtf (Rich Text Format) (nessun immagine o oggetti incorporati sono visualizzati). Specifichi il nome del file. Assicurati che il file esista. Dovresti o distribuirlo con il gioco o includerlo nel gioco tramite Include Files nel menu Resources (risorse). (Nota che nessuna barra di scorrimento verrà mostrata nè c'è un altro modo per scorrere il testo. Quindi il testo deve adattarsi alla finestra.) Questa azione è disponibile solo nell'Edizione Standard.

Splash Image
Con questa azioni puoi mostra un file immagine. I più comuni tipi di file immagine sono supportati. Specifichi il nome del file. Assicurati che il file esista. Dovresti o distribuirlo con il gioco o includerlo nel gioco tramite Include Files nel menu Resources (risorse). Puoi ad esempio usare questo per creare una serie di immagini come aiuto per il tuo gioco. Non includendo le immagini nel gioco stesso ridurrà le dimensioni del gioco e il tempo di caricamento del gioco. Questa azione è disponibile solo nell'Edizione Standard.

Splash Webpage
Con questa azione puoi mostrare una pagina web. Fornisci l'URL e se dovrebbe essere mostrato nella finestra del gioco o nel web browser prefedinito. L'URL può essere l'indirizzo di un sito web, per esempio http://www.yoyogames.com) o può essere il nome del file per un file html locale. Nota comunque che devi fornire il percorso completo per questo per funziona ( che non è il caso di altre azioni splash). Quindi usa meglio come url qualcosa come "" + working_directory + "\index.html". (I primi due quotes sono richiesti per indicare che questa è un'espressione piuttosto che una stringa). Dovresti o distribuirlo con il gioco o includerlo nel gioco tramite Include Files nel menu Resources (risorse). Puoi ad esempio usare questo per creare pagine d'aiuto di fantastia per il tuo gioco. Questa azione è disponibile solo nell'Edizione Standard.

Splash Video
Co questa azione puoi visualizzare un file video. Specifichi il nome del file e quando deve essere ciclato. Assicurati che il file video esista. Dovresti o distribuirlo con il gioco o includerlo nel gioco tramite Include Files nel menu Resources (risorse). Questa azione è disponibile solo nell'Edizione Standard.

Splash Settings
Con questa azione puoi cambiare certe preferenze per le schermate splash che mostri con le precedenti azioni. Puoi fornire la didascalia per la finestra d'aiuto (solo quando è mostrata in una finestra normale separata), puoi indicare dove le schermate splash devono essere visualizzate (dentro la finestra del gioco, in una finestra normale, o schermo pieno), puoi indicare quando il pulsante di chiusura deve essere mostrato nell'angolo a destra in alto della schermata splash e quando premendo il pulsante escape o cliccando con il mouse nella schermata splash la chiuderà. (Per le pagine web cliccando con il mouse non funzionerà nè il tasto escape a meno che il tasto di chiusura è visualizzato.) Questa azione è disponibile solo nell'Edizione Standard.

Replace Sprite
Questa azione può essere usata per sostituire uno sprite dai contenuti di un file. Indichi lo sprite che vuoi sostituire, il nome del file e il numero di sottoimmagini nello sprite. Molti formati di immagine diversi sono supportati, ad esempio .bmp, .jpg, .tif, and .gif. Per un file gif il numero di sottoimmagini è automaticamente deciso, basato sul numero di sottoimmagini nel file gif. Altre impostazioni per lo sprite, ad esempio quando è trasparente o no, non sono cambiate. Puoi usare questa azione per evitare di memorizzare tutte le sprites nel programma stesso. Per esempio, all'inizio di un livello puoi sostituire le sprites degli attuali character sprites che vuoi usare. NON cambiare uno sprite che è al momento usato in un'istanza nella room. Questo potrebbe dare effetti indesiderati con le collisioni. Questa azione è disponibile solo nell'Edizione Standard.

Replace Sound
Con questa azione puoi sostituire un suono dai contenuti di un file (.wav, .mid, or .mp3). Specifichi il suono e il nome del file. Questo evita di memorizzare tutti i suoni nel gioco stesso. Per esempio, puoi usare diversi pezzi di musica di sottofondo e scegliere quale vuoi suonare. NON cambiare un suono mentre è in riproduzione. Questa azione è disponibile solo nell'Edizione Standard.

Replace Background
Con questa azione puoi sostituire uno sfondo dai contenuti di un file. Molte formati di immagini diversi sono supportati, ad esempio, .bmp, .jpg, .tif, and .gif. Specifichi il background e il nome del file. Questo evita di memorizzare i backgrounds nel gioco stesso. NON cambiare lo sfondo se è visibile. Questa azione è disponibile solo nell'Edizione Standard.
