Costanti

Questa funzionalità è disponibile solo nella Standard Edition di GameMaker.

Nella modalità avanzata puoi definire delle costanti che puoi usare negli script e nei valori delle azioni. Ogni costante ha un nome e un valore. I nomi devono seguire le stesse regole delle variabili, quindi, devono iniziare con una lettera o con un trattino basso "_". Rendi le costanti facilmente distinguibili. Una convezione comune è quella di usare soltanto lettere maiuscole e trattini.

Per definire delle costanti devi scegliere Define Costants dal menu Resources. Verrà mostrata la seguente finestra (con costanti definite):

In alto puoi vedere le costanti. A sinistra c'è il nome e a destra il valore. Puoi selezionare il nome o il valore con il mouse e cambiarlo scrivendo. Il valore di una costante dev'essere un'espressione costante. Quindi, un numero o una stringa (con virgolette attorno ad essa) costante, oppure un'espressione. L'espressione viene valutata prima di qualunque altra cosa nel gioco. Quindi non può per esempio riferirsi alla room corrente, alle istanze o agli script. Ma può contenere le costanti built-in e i nomi delle risorse.

Puoi appendere una costante alla fine di una lista usando il bottone Add (o premendo Invio) e cancellarla usando Delete. Puoi anche inserire una costante sopra alla costante selezionata. Ci sono inoltre dei bottoni per cancellare tutte le costanti, spostarle, e ordinarle per nome

Infine puoi salvare tutte le costanti o caricarle da un file di testo. Questo è utile quando vuoi salvare delle collezioni di costanti utili per usarle dopo. Quando carichi delle costanti, queste vengono aggiunte alla lista corrente. Il file di testo può anche essere modificato usando un editor di testo. Ogni linea contiene una definizione nel formato NOME=VALORE.

È una buona pratica mettere ogni valore che usi più volte in una costante. Inoltre, se non sei completamente sicuro riguardo a un certo valore è meglio se lo metti un una costante in modo tale da poterlo cambiare facilmente.
