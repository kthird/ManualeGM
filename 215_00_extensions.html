Pacchetti estensione

Questa funzionalità è disponibile solo nell'edizione Standard di GameMaker.

I pacchetti estenzione estendono le possibilità di GameMaker. Un pacchetto estensione può aggiungere un insieme di azioni a GameMaker o aggiunge un numero di funzioni aggiuntive e costanti al linguaggio GML integrato in GameMaker. Quando i pacchetti estensione sono disponbili per te, l'aiuto su di essi è posto nel menu Help (aiuto).

Quando fai doppio click su Extension Packages (pacchetti estensione), la voce di risorsa nel seguente modulo è visualizzato:

In questo modulo puoi indicare quale pacchetto estensione deve essere usato nel gioco. Sulla sinistra c'è una lista di pacchetti Usati e sulla destra è la lista di pacchetti Disponibili. Per spostare un pacchetto da una lista all'altra, semplicemente selezionalo e premi il pulsante in mezzo alla lista. Quando selezionato un pacchetto una piccola descrizione è data sotto. Ulteriore aiuto è disponibile premendo il pulsante Help.

I pacchetti estensione sono una parte estremamente potere di GameMaker. Alcuni pacchetti estensione sono provvisti con GameMaker ma molti altri saranno disponibili attraverso il sito web. Per installare nuovi pacchetti o disinstallarne uno esistente, premi il pulsante Install (installa). Il seguente modulo è visualizzato:

Vedrai una lista di tutti i pacchetti estenzione installati. Selezionando un pacchetto e cliccando il pulsante Uninstall (disinstalla) il pacchetto è rimosso dal sistema.

I pacchetti installazione sono distribuiti nella forma di un .gex file. Puoi trovare un numero di tali paccketti sul nostro sito web in qui potrai andare premendo il pulsante Find More (cerca altro). Una volta scaricato tale file di paccheto sul tuo computer, premi il pulsante Install (installa) e seleziona il file di pacchetto. Verrà installato nel sistema.

Se vuoi creare i tuoi pacchetti estensione, per favore controlla l'informazione che è dispobilie su http://www.yoyogames.com/extensions.
